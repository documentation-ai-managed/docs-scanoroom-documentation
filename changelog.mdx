## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added support for outdoor scanning with improved stabilization
- Introduced batch export functionality for multiple projects
- Enhanced measurement tools with area and volume calculations

## Improvements

- Improved LiDAR processing speed by 40%
- Better handling of complex indoor environments
- Enhanced user interface for tablet devices

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete redesign with modern iOS interface
- Added cloud synchronization across devices
- Introduced API for third-party integrations
- New export formats including FBX and STL

## Breaking Changes

- Minimum iOS version increased to 15.0
- API endpoints restructured for better consistency

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added annotation tools for marking important features
- Introduced floor plan generation from 3D scans
- New sharing options with permission controls

## Bug Fixes

- Fixed scanning interruptions on low battery
- Resolved issues with large room scanning
- Improved accuracy of distance measurements

</Update>
